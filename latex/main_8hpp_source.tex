\DoxyCodeLine{30 \textcolor{comment}{ * \# Background}}
\DoxyCodeLine{31 \textcolor{comment}{ *}}
\DoxyCodeLine{32 \textcolor{comment}{ * Zero-\/dimensional (0D) models}}
\DoxyCodeLine{33 \textcolor{comment}{ * are lightweight methods to simulate bulk hemodynamic quantities in the}}
\DoxyCodeLine{34 \textcolor{comment}{ * cardiovascular system. Unlike 3D and 1D models, 0D models are purely}}
\DoxyCodeLine{35 \textcolor{comment}{ * time-\/dependent; they are unable to simulate spatial patterns in the}}
\DoxyCodeLine{36 \textcolor{comment}{ * hemodynamics. 0D models are analogous to electrical circuits. The flow rate}}
\DoxyCodeLine{37 \textcolor{comment}{ * simulated by 0D models represents electrical current, while the pressure}}
\DoxyCodeLine{38 \textcolor{comment}{ * represents voltage. Three primary building blocks of 0D models are resistors,}}
\DoxyCodeLine{39 \textcolor{comment}{ * capacitors, and inductors Resistance captures the viscous effects of blood}}
\DoxyCodeLine{40 \textcolor{comment}{ * flow, capacitance represents the compliance and distensibility of the vessel}}
\DoxyCodeLine{41 \textcolor{comment}{ * wall, and inductance represents the inertia of the blood flow. Different}}
\DoxyCodeLine{42 \textcolor{comment}{ * combinations of these building blocks, as well as others, can be formed to}}
\DoxyCodeLine{43 \textcolor{comment}{ * reflect the hemodynamics and physiology of different cardiovascular}}
\DoxyCodeLine{44 \textcolor{comment}{ * anatomies.These 0D models are governed by differential algebraic equations}}
\DoxyCodeLine{45 \textcolor{comment}{ * (DAEs).}}
\DoxyCodeLine{46 \textcolor{comment}{ *}}

\DoxyCodeLine{49 \textcolor{comment}{ * \#\# Model}}
\DoxyCodeLine{50 \textcolor{comment}{ *}}
\DoxyCodeLine{51 \textcolor{comment}{ * The solver uses a highly modular framework to model the vascular anatomy,}}
\DoxyCodeLine{52 \textcolor{comment}{ * using individual classes to represent different 0D elements of the}}
\DoxyCodeLine{53 \textcolor{comment}{ * model. The elements are part of the MODEL namespace. Currently}}
\DoxyCodeLine{54 \textcolor{comment}{ * supported elements are:}}
\DoxyCodeLine{55 \textcolor{comment}{ *}}
\DoxyCodeLine{56 \textcolor{comment}{ * * MODEL::BloodVessel: RCL blood vessel respresentation with optional}}
\DoxyCodeLine{57 \textcolor{comment}{ * stenosis.}}
\DoxyCodeLine{58 \textcolor{comment}{ * * MODEL::Junction: Junction element with arbitrary inlets and outlets.}}
\DoxyCodeLine{59 \textcolor{comment}{ * * MODEL::FlowReferenceBC: Prescribed flow boundary condition.}}
\DoxyCodeLine{60 \textcolor{comment}{ * * MODEL::PressureReferenceBC: Prescribed pressure boundary condition.}}
\DoxyCodeLine{61 \textcolor{comment}{ * * MODEL::ResistanceBC: Resistance boundary condition.}}
\DoxyCodeLine{62 \textcolor{comment}{ * * MODEL::WindkesselBC: RCR Windkessel boundary condition.}}
\DoxyCodeLine{63 \textcolor{comment}{ * * MODEL::OpenLoopCoronaryBC: Open Loop coronary boundary condition.}}
\DoxyCodeLine{64 \textcolor{comment}{ *}}
\DoxyCodeLine{65 \textcolor{comment}{ * The elements are based on the parent MODEL::Block class. More information}}
\DoxyCodeLine{66 \textcolor{comment}{ * about the elements can be found on their respective pages. The elements are}}
\DoxyCodeLine{67 \textcolor{comment}{ * connected to each other via nodes (see MODEL::Node). Each node corresponds to}}
\DoxyCodeLine{68 \textcolor{comment}{ * a flow and a pressure value of the model. The MODEL::DOFHandler handles the}}
\DoxyCodeLine{69 \textcolor{comment}{ * degrees-\/of-\/freedom (DOF) of the system by assigning DOF indices to each}}
\DoxyCodeLine{70 \textcolor{comment}{ * element that determine the location of the local element contribution in the}}
\DoxyCodeLine{71 \textcolor{comment}{ * global system. The complete model is stored and managed by the MODEL::Model}}
\DoxyCodeLine{72 \textcolor{comment}{ * class.}}
\DoxyCodeLine{73 \textcolor{comment}{ *}}
\DoxyCodeLine{74 \textcolor{comment}{ * \#\# Algebra}}
\DoxyCodeLine{75 \textcolor{comment}{ *}}
\DoxyCodeLine{76 \textcolor{comment}{ * Everything related to solving the system of equation is contained in the}}
\DoxyCodeLine{77 \textcolor{comment}{ * ALGEBRA namespace. The svZeroDSolver provides two different strategies for}}
\DoxyCodeLine{78 \textcolor{comment}{ * handling the system of equation: a dense strategy (see ALGEBRA::DenseSystem)}}
\DoxyCodeLine{79 \textcolor{comment}{ * and a sparse strategy (ALGEBRA::SparseSystem). The ALGEBRA::Integrator}}
\DoxyCodeLine{80 \textcolor{comment}{ * handles the time integration scheme (see Integrator documentation for}}
\DoxyCodeLine{81 \textcolor{comment}{ * more information on that).}}
