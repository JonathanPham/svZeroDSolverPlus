\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_m_o_d_e_l}{MODEL}} namespace. 

The \mbox{\hyperlink{namespace_m_o_d_e_l}{MODEL}} namespace contains all classes that are used to build a 0D blood flow model, including the components (elements) of the model themselves and a few auxilliary classes.
